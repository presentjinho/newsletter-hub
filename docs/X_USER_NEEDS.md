# X에서 보이는 뉴스레터·인박스 수요 (2025–2026 요약)

조사: Proton Mail 구독 대시보드, Unroll.Me 롤업, HEY Feed, 필터/주말 읽기, bulk unsubscribe 담론 등.

## 사람들이 반복해서 원하는 것

| 니즈 | 제품 대응 (오늘의 편지함) |
|------|---------------------------|
| 구독을 **한 화면**에서 보기 | 구독 관리 대시보드 `#subscriptions` |
| 일괄 정리 / 해지 표시 | 다중 선택 → 구독 중·나중에·해지함 |
| 주중에 쌓고 **주말에만** 읽기 | 상태 `나중에` + 주말 몰아보기 MD |
| 인박스 롤업 한 통 | 주말 몰아보기 Markdown 내보내기 |
| CSV/내보내기 | 구독 CSV, OPML, 메모 MD, 전체 백업 JSON |
| 해지가 쉽다는 **안심** | 카드·대시보드 해지 안내 (대행 아님) |
| 검색·필터로 줄이기 | 검색, 상태 필터, 필터 초기화 |
| 데이터 내 것 | localStorage + 백업/복원 |

## 의도적으로 안 하는 것

- 타사 메일 **대행 해지** (법적·신뢰 리스크)
- 메일함 본문 스크래핑
- 강제 iframe 미리보기 (보안 정책)

## 참고 시그널

- Proton: “newsletter subscriptions in one dashboard”
- Unroll.Me: bulk unsubscribe + roll-up
- HEY: long newsletter collapse / Feed
- 개인 생산성 팁: unsubscribe 필터, Saturday read pile
