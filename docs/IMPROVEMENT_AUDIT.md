# 개선 점검 (2026-07-15)

## 번역 중도 끊김 — 원인

1. 청크 실패 시 **그 구간을 버리고** 다음으로 넘어감 → 본문 중간 공백/끊김
2. 최대 20청크(약 8천자) 하드캡 후 뒤는 “생략”만 표시
3. MyMemory/Lingva 지역·쿼터 실패가 잦음 (429 / region / 403)

## 적용한 수정

- 실패 청크 = **원문 유지** (끊기지 않음)
- 연속 실패 시 나머지 원문 붙이고 종료 / 전부 실패면 페이지 번역 모드
- 성공 엔진 세션 기억 + sessionStorage 캐시
- **페이지 번역** 모드 (Google 페이지 단위, API보다 안정)
- 문단·문장 기준 분할로 잘림 품질 개선
