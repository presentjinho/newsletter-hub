# 모음집 재사용 정책

목적: 이 디렉터리는 독자가 원 출처로 안전하게 이동하도록 돕는다. `무료 공개`는 `원문·이미지 재배포 가능`과 같은 뜻이 아니다.

## 등록 등급

### A. 원문 외부 링크만

대상: 일반 뉴스 사이트, 공영방송, 비영리 매체, 일반 뉴스레터.

- 표시 가능: 출처명, 공식 URL, 운영 국가·분야·발행 빈도 같은 사실 정보, 운영자가 새로 쓴 1~2문장 한국어 소개.
- 표시 금지: 원문 전문, 기사 본문 번역, 원본 이미지·로고·스크린샷, 유료 콘텐츠 우회본.
- 원칙: 제목은 가능한 한 출처명/서비스명 중심으로 쓰고, 요약은 독자적으로 작성한다.

### B. 조건부 재사용

대상: 개별 글마다 Creative Commons 또는 명시적 재게시 조건이 붙은 출처.

- 글마다 라이선스와 예외를 다시 확인한다. 사이트 전체가 무료여도 자동 승인하지 않는다.
- 저자·원 출처·원문 URL·라이선스 링크를 함께 표시한다.
- 번역·편집은 해당 라이선스의 변경 허용, 동일조건·비영리 조항을 확인한 뒤에만 한다.

### C. 명시 오픈 라이선스·퍼블릭 도메인

대상: 명시적으로 재사용 허용된 데이터·시각화·미디어.

- World Bank Open Data: 해당 데이터셋의 CC BY 4.0 표기와 귀속·변경 표시를 지킨다.
- Our World in Data: 각 차트·데이터셋의 라이선스를 확인하고 출처와 라이선스를 표시한다.
- NASA: NASA SVS 등 `public domain unless otherwise noted`인 자료만 사용한다. NASA 로고·휘장, 제3자 권리, NASA가 보증하는 것처럼 보이게 하는 사용은 제외한다.

## 등록 전 필수 증거

1. 공식 발행 페이지 또는 공식 RSS/뉴스레터 URL
2. 최신 발행 또는 갱신 증거
3. 재사용 등급(A/B/C)
4. B/C인 경우 라이선스 원문 URL과 적용 대상(글·데이터셋·미디어)
5. 국가, 분야, 언어, 광고·유료 여부

## 운영 규칙

- 라이선스 증거가 없으면 기본값은 A(외부 링크만)다.
- 로고·썸네일은 허가 또는 라이선스가 확인되기 전까지 저장·복제하지 않는다.
- 자동 번역은 이 사이트가 직접 쓴 소개와 메타데이터에만 적용한다. 제3자 기사·뉴스레터 본문 자동 번역본은 저장·공개하지 않는다.
- 삭제·정정 요청은 즉시 `확인 필요`로 내리고, 근거 확인 전까지 재게시하지 않는다.
- 법률 자문이 아니라 운영 기준이다. 국가별 저작권·상표권·데이터베이스권은 별도 검토한다.
