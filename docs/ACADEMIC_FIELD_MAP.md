# 학문 분야 전체 분류와 공개 피드 등록 지도

목적: 뉴스레터 허브가 생활 관심사뿐 아니라 학문 분야로도 탐색되게 한다. 아래 분류는 대학 전공명 그대로가 아니라, 독자가 피드·뉴스레터를 고르는 **탐색용 태그 체계**다.

공통 원칙: 모든 일반 콘텐츠는 `A`(공식 링크·자체 소개만)다. 개별 콘텐츠에 재사용 근거가 있는 경우만 `B/C`로 올린다. 기준은 [REUSE_POLICY.md](REUSE_POLICY.md)를 따른다.

## 카탈로그 입력 규칙

각 카드에 상위 분야와 세부 태그를 모두 넣는다.

```text
disciplineGroup: 자연과학
discipline: 지구·해양·우주
subjects: [기후, 해양, 재난]
country: 미국
feedType: RSS
reuseLevel: A
```

- 한 출처에 여러 분야가 있으면 `disciplineGroup` 하나와 `subjects` 여러 개를 둔다.
- “학술지”, “프리프린트”, “데이터셋”, “뉴스레터”, “RSS”는 분야가 아니라 `type/feedType`으로 분리한다.
- 의학·교육·농업·법 등은 위 네 큰 축과 겹치므로 검색 태그에서는 독립적으로 노출한다.
- 이 분류는 학과 서열이나 콘텐츠 품질 등급이 아니다. 공식 피드·라이선스 증거가 있는지를 별도로 검증한다.
