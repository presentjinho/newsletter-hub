# 국가·분야별 공공·비영리 공개 피드 카탈로그

기준일: 2026-07-15. 이 문서는 **공식 RSS 또는 뉴스레터가 확인된 공영·정부·비영리·공개 데이터 출처**만 모은 1차 시드다.

표기:

- `A`: 이 사이트에는 출처명, 공식 링크, 운영자가 직접 쓴 한국어 소개만 표시. 원문·이미지·번역문 복제 금지.
- `C 후보`: 출처 또는 개별 자료에 명시 라이선스가 있을 때만 재사용 가능. 실제 카드에는 라이선스 URL과 적용 범위를 붙인 뒤 `C`로 승격.
- RSS/뉴스레터 URL은 언제든 바뀔 수 있으므로 배포 전 링크 검사한다.

## 등록 순서

1. 이 목록의 공식 구독/피드 페이지를 카드의 `url`로 쓴다.
2. `country`, `category`, `language`, `type`, `frequency`를 확인해 입력한다.
3. `reuseLevel`은 기본 A로 설정한다.
4. C 후보만 `licenseUrl`과 재사용 대상(데이터·차트·사진 등)을 함께 기록한다.
5. 30일 이내 새 항목이 확인되지 않거나 RSS/뉴스레터가 종료되면 `확인 필요`로 내린다.

자세한 법적 경계는 [REUSE_POLICY.md](REUSE_POLICY.md)를 따른다.
